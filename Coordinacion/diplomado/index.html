# Crear Diplomado

Consejo

Antes de crear un Diplomado nuevo, debes asesorarte de que los alumnos que lo componen estén creados antes.

Al crear el Diplomado, debes asignarle un nombre y preocuparte de rellenar todos los campos solicitados. Como último paso, debes seleccionar los cursos que componen el Diplomado.

Puedes mirar el video para saber los pases de una creación de Diplomado.

Opcionalmente se le puede cargar una imagen de portada al Diplomado.

# Edición del Diplomado

En primera instancia, debes seleccionar el Diplomado que quieres editar.
