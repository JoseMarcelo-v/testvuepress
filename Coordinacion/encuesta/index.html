# Crear Encuesta

Dale un nombre a la encuesta y presiona el botón. Seguidamente después de crear la encuesta puedes agregarle preguntas. Sigue los pasos del video.

Consejo

Puedes asignar la encuesta a distintos cursos.

# Listado Encuesta

Se despliega una lista de todas las encuestas existentes.

Consejo

Las encuestas tienen dos estados (revisión – publicado). Revisión es cuando estás agregando preguntas y respuestas a la encuesta y, una vez terminada, procederás a publicarla para que se muestre en los cursos a los que la asignaste.

# Agregar Preguntas

A continuación, te presentamos los pasos para agregar preguntas a la encuesta.

# Eliminar Preguntas

Simplemente presiona el botón del icono de basurero para que la pregunta quede omitida y no se despliegue en la encuesta, tal como se muestra a continuación.
