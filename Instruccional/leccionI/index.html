# Crear Lección

Para crear una lección primero debes buscar el curso en cuestión. Baja hasta llegar a la sección Creación Material o Módulo y pincha el recuadro para agregarla.

Importante

Debes saber si el curso es simple o compuesto antes de crear la lección.

Después de pinchar el recuadro, selecciona Contenido Material y presiona el botón Crear Nuevo Contenido.

# Crear Lección HTML

Para crear una lección HTML debes pinchar en Ir al detalle, tal como aparece en la siguiente imagen.

En la siguiente vista deberás seleccionar Lección Html.

A continuación, debes agregar los contenidos que desees. Recuerda guardar siempre los cambios. Puedes seguir el ejemplo del video de abajo para más detalles.

Si necesitas crear una lección o clase de tipo “zoom”, debes siempre elegir el tipo de lección HTML. Con esto el área de Servicios podrá cargar los parámetros para generar una clase por Zoom.

# Cambiar Posición Lección

Para cambiar de posición la lección, sólo basta con seleccionarla y arrastrarla hacia donde desees, tal como se muestra en el video.

# Editar Datos Lección

Selecciona "Ir A Detalle" como en la imagen.

Selecciona "Leccion Normal".

Luego, modifica los campos que sean necesarios y recuerda siempre guardarlos. A continuación, encontrarás un video a modo de ejemplo.

# Crear Link S3

Para crear un Link en s3 debes estar dentro de la edición de datos de lección y seleccionar "Generador Link s3".

Luego debes seleccionar el archivo que deseas subir a s3.

Al presionar el botón Crear Link se generará un link con el archivo ya subido a s3, el cual debes copiar y pegar, tal como lo muestra la imagen.

Recuerda guardar los cambios. Para dudas, puedes revisar el video que está aquí abajo.

# Crear Foro Lección

Para la creación de foro debes estar adentro del detalle de la lección y pinchar Agregar Foro.

Se desplegará una modal con los campos requeridos para poder agregar un foro a la lección.

Debes agregar las instrucciones, la pregunta y la cantidad de caracteres que admite el foro.

A continuación, puedes ver el video que lo ejemplifica.

# Crear Recurso Lección

Para la creación de un recurso debes pinchar sobre Agregar Nuevo Recurso, tal como lo muestra la imagen.

Después, debes escribir el nombre del curso y seleccionar un archivo como, por ejemplo, un PDF. Recuerda siempre guardar los cambios.

# Eliminar Lección

Para eliminar una lección debes pinchar Eliminar Lección.

Se desplegará una modal, la cual te pedirá escribir el motivo de la eliminación de la lección.

Puedes ver el video de ejemplo.
