# Crear Módulo

Para crear un nuevo módulo debes pinchar sobre el recuadro de ingreso de contenido, tal como se muestra a continuación.

Después, debes seleccionar Contenido Módulo y presionar el botón Crear Nuevo Contenido, como se muestra aquí abajo.

Video de ejemplo del paso a paso.

# Eliminar Módulo

Para eliminar el módulo creado sólo tienes pinchar Eliminar Lección, tal como se muestra a continuación.

Se desplegará una modal, en donde debes escribir el motivo de la eliminación del módulo, tal como se muestra abajo.

Video de ejemplo del paso a paso.

# Cambiar Posición Módulo

Para cambiar la posición del módulo sólo hay que pinchar y arrastrar el módulo a la posición que desees. A continuación, verás un video del paso a paso.

# Cambiar Información del Módulo

Para cambiar o editar la información del módulo, debes pinchar Ir al Detalle, tal como se muestra en la imagen.

Tendrás los campos disponibles para su edición, como se muestra a continuación.

Video de ejemplo del paso a paso.

# Crear Lección Módulo

Dentro del detalle del módulo podrás crear lecciones pinchando Cargar Nueva Lección, como se muestra a continuación.

Se desplegará una modal donde tendrás que poner un nombre para la lección, como se muestra a continuación.

Video de ejemplo del paso a paso.

# Cambiar Posición Lección Módulo

Para cambiar la posición de las lecciones dentro del módulo, debes pinchar la lección y arrastrarla al lugar que desees. Ahora puedes ver el video de ejemplo paso a paso.

# Eliminar Lección

Para eliminar una lección de un módulo pincha Eliminar Lección, tal como se muestra a continuación.

Se desplegará una modal para la confirmación de eliminar la lección, como lo muestra la imagen.

Se desplegará una modal para la confirmación de eliminar la lección, como lo muestra la imagen.

# Crear Control Módulo

Para la creación de controles dentro del módulo, debes pinchar Agregar Control, tal como se muestra abajo.

Se desplegará una modal para asignarle un nombre al control.

Automáticamente el control se agregará a la lista de Lecciones Asociadas al Módulo.

Ahora puedes ver el video de ejemplo paso a paso.

# Eliminar Control Módulo

Para eliminar el control del módulo pincha Eliminar Control, tal como lo muestra la imagen.

Se desplegará una modal de confirmación, en la cual debes agregar el motivo de la eliminación del control.

Ahora puedes ver el video de ejemplo paso a paso.
